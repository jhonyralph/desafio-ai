# Web App - Challenge Shell

Aplicação Next.js para o desafio técnico.

## Comandos

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

Sem módulo de negócio pré-configurado.
Os módulos serão criados sob demanda em `src/modules/<nome-do-modulo>/`.

## Variáveis esperadas

Criar `.env.local`:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:3001
```
